PubSearch

PubSearch is a literature annotation tool for biological data hosts. It provides a simple user interface and powerful features. The PubSearch database is designed to contain the complete literature information of a model organism, along with the controlled vocabularies, gene and sequence information, and other data.

PubSearch is being developed at the Arabidopsis Information Resource (TAIR) and is part of the Generic Model Organism Database (GMOD) Construction Set.

PubSearch is part of a suite of programs that include PubFetch and PubTrack.
